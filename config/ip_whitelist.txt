# IP Whitelist
# Bu dosya whitelist'e eklenecek IP adreslerini içerir
# Her satırda bir IP adresi veya CIDR notasyonu olabilir
# Yorum satırları # ile başlar

# Örnek: Tek IP adresi
# 192.168.1.100

# Örnek: CIDR network (tüm subnet)
# 192.168.1.0/24

# Örnek: IPv6
# 2001:db8::/32

# Production'da kullanmak için yukarıdaki örnekleri kaldırın ve gerçek IP'leri ekleyin
